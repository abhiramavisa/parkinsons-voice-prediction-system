# 🧠 Parkinson’s Disease Prediction using Voice

A Machine Learning web application that predicts Parkinson’s disease using voice recording analysis.

## 🚀 Features
- Upload voice recording (.wav)
- Automatic feature extraction
- ML model prediction
- Confidence threshold display
- Clean web interface

## 🛠 Tech Stack
- Python
- Flask
- Scikit-learn
- Librosa
- HTML/CSS

## 🧠 Model
- Trained on Parkinson’s voice dataset
- Scaled using StandardScaler
- Threshold optimized
